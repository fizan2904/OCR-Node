About this project

This project is ment to convert images, pdf or any document to a soft copy so as to reduce the workload of typing all the documents using Tesseract.Its advanteges are

  • To scan any document on the go
  • Reduce the workload of typing every document
  • Api is also available for integrating into other websites or projects
  • It is very light weight and fast as it is not built on python but on Nodejs
